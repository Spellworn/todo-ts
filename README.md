todo-ts - список задач (React + TypeScript + Vite)

Небольшое приложение «To‑Do» для демонстрации использования React, TypeScript и Vite.

Коротко:
- Добавление, редактирование и удаление задач
- Сортировка задач по их статусу и поиск по названию
- Компонентная архитектура с использованием Redux Toolkit
- Стили реализованы через CSS Modules

Технологии:
- React 18
- TypeScript
- Vite (dev server, сборка)
- Redux Toolkit
- CSS Modules
